I choose this project because loan processing is slow, manual, and prone to human need. 
Banks need a way to quickly filter out applications.
In the Indian economy, loans leads to growth and so are Important.
My Approach:
This code helps makes decisions based on the data and makes sure that deserving candidates with greater CIBIL score and income get money.
I used Logistic Regression,it uses binary outcomes(0,1) because it is fast, explainable, and provides a probability score rather than just a Yes or a No.
Tools:
Used Pandas for data management and Logical Regression for the machine learning.
Decisions I made:
I focused on Income and Credit History (CIBIL) used in India, these are the two important factors for any loan provided.
I encoded categorical data (Good/Bad) into Binary (1/0) so the mathematical function could process it.
Challenges Faced:
1)Since I used a small manual dataset, the model initially had a risk of Overfitting.
2)I had to use an try-except block because if a user entered "High" instead of "1,it could break.
Summary:
Through this project, I successfully transitioned from writing basic logic to building a good project which is useful.
